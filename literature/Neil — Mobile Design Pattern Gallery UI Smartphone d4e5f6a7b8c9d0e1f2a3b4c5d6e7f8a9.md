# One Sentence
Catalogs 90+ named mobile app UI patterns across 11 categories, illustrated by 1,000+ screenshots from iOS, Android, and Windows Phone apps.

---

# More Sentences
Neil organizes mobile patterns into 11 chapters: Navigation, Forms, Tables and Lists, Search, Sort and Filter, Tools and Actions, Charts, Invitations, Feedback and Affordance, Help, and Anti-Patterns. Each pattern has a name, "best uses," and annotated screenshots. The book is the primary academic and practitioner reference for named mobile UI patterns, widely cited in mobile HCI research. No formal empirical validation is provided.

---

# Key Points

### 11 pattern categories, 90+ named patterns
Categories span the full range of mobile screen interactions; the finest-grained mobile catalog available.

### Richly illustrated with real apps
1,000+ screenshots provide grounding in actual app implementations, making patterns highly concrete.

### Anti-patterns chapter
Includes a dedicated anti-patterns chapter — negative examples of what not to do. Rare among pattern catalogs.

# Other Notes
This is the most detailed mobile-specific pattern catalog and the closest precedent to PatternGenUI's mobile-first focus. The 11-category structure could inform PatternGenUI's taxonomy. Currency is limited (2014).

# Take-Away
- The anchor mobile pattern catalog: any PatternGenUI pattern library should be reconciled against Neil's taxonomy.
- The 11-category structure is a useful starting scaffold for a mobile-first pattern specification language.
