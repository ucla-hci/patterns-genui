When input or feedback is needed from the user, end with a direct question — not a list of things to react to.

**Why:** User finds the "list of things to react to" format passive and verbose; a question is clearer and more actionable.

**How to apply:** Any time a checkpoint or decision point requires user input, close with a single focused question. If multiple things need input, pick the most important one and ask that first.
