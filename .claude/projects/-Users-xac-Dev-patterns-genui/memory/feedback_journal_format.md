Use bullets or short labeled paragraphs (e.g., **Label.** text) when writing `###` result sections in `.agent/journal.md`. Avoid dense single-paragraph prose.

**Why:** User found single-paragraph results hard to scan.

**How to apply:** Every time a journal result section is written, structure it with bullets or short paragraphs — never a single wall of text.
